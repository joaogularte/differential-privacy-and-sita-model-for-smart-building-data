# Aplicação de _Differential Privacy_ em dados de _smart building_ seguindo o modelo SITA

Este repositório contém os _datasets_, o _notebook_ Jupyter e os resultados do experimento ilustrado abaixo.

![Etapas do experimento](images/experimento.png "Etapas do experimento")

Este experimento tem como objetivo analisar o uso e o impacto que a técnica de _Differential Privacy_, seguindo o modelo SITA, exerce sobre o funcionamento dos serviços de prédios inteligentes.
